## Objective

Prepare and submit the X-Ray Vision Diagnostics safety documentation to the SAE J1742 review panel. SAE J1742 ("Recommended Practice for High Voltage Wiring, Connections, and High Voltage Component Identification") is the primary EV safety standard referenced by OSHA and OEM service networks. Achieving J1742 review sign-off is a prerequisite for Phase 4 commercial launch and for Toyota/Ford OEM endorsement.

**Important distinction:** J1742 does not certify software tools. What the review panel evaluates is whether the system's guidance, warnings, and HV component identification are consistent with J1742's requirements. The output is a letter of no objection (or a list of required changes) — not a formal "certification."

---

## SAE J1742 Key Requirements Relevant to This Product

| J1742 Clause | Requirement | X-Ray Vision Diagnostics Impact |
|---|---|---|
| §4.1 | HV components identified with orange color coding | CAD HV bus layer must use orange per standard |
| §4.2 | HV connectors identified with warning labels | CAD layer must display warning label overlays on HV connectors |
| §5.1 | Minimum working clearance from HV components: 300mm | HV-STOP zone radius set at 300mm — compliant |
| §5.3 | Service personnel must verify HV bus discharge before contact | HV-STOP log must record discharge verification step |
| §6.1 | Written HV service procedures required for each model | Per-model lockout/tagout procedures (per OSHA review, `docs/compliance/osha_audit_readiness.md`) |
| §7.2 | Training required for personnel servicing HV systems | Onboarding program includes HV awareness module (`docs/deployment/onboarding_program_spec.md`) |
| §8.1 | Automatic de-energization on crash detection | Out of scope for service tool (applies to vehicle system, not diagnostic tool) |

## Submission Package Contents

### Section 1: Product Description
- What X-Ray Vision Diagnostics does and does not do
- Clear statement: system is a diagnostic aid; it does not modify or control HV components
- System architecture overview (agents, edge server, headset)
- Scope: enrolled vehicle models and fault types

### Section 2: HV Component Identification Compliance
- Screenshots of CAD orange HV bus layer on Toyota bZ4X and Ford Mach-E
- Confirmation that orange color matches SAE J1742 §4.1 specification (Pantone 021 C, RAL 2009)
- Confirmation that HV connector warning label overlays are sourced from OEM CAD data (not generated by the system)

### Section 3: Working Clearance Implementation
- Technical description of HV-STOP proximity detection system
- Sensor placement diagram (capacitive + thermal camera)
- 300mm zone radius implementation: how it is calculated and enforced
- Latency spec: < 80ms activation (p95) — with test data from `docs/qa/hv_stop_safety_audit.md`

### Section 4: Discharge Verification Protocol
- Full HV isolation procedure per vehicle model
- HV-STOP log format showing discharge verification field
- Demo: system UI flow from HV-STOP trigger to discharge verification to resume

### Section 5: Training Program Summary
- Onboarding program description (3-hour orientation)
- HV awareness training module content outline
- Certification issuance process

### Section 6: Audit Log Samples
- 10 sample HV-STOP log entries from Phase 2 QA audit (`docs/qa/hv_stop_safety_audit.md`)
- Demonstrating: timestamp, employee ID (encrypted), vehicle VIN, voltage, activation latency, acknowledgement

### Section 7: OEM Validation Statement
- Letter from Toyota and Ford confirming: CAD data used by the system is OEM-sourced and OEM-approved
- Statement that repair steps displayed by DiagnosticCore are OEM-sourced (no LLM-generated instructions)

## Submission Process

### Step 1: Pre-Submission Consultation (Month 10)
- Request informal meeting with SAE J1742 panel chair
- Present product overview; identify any pre-existing concerns
- Confirm submission format requirements

### Step 2: Package Assembly (Month 11)
- Compile all 7 sections above
- Legal review: outside counsel confirms no material gaps
- Engineering sign-off: Engineering Lead certifies technical claims are accurate

### Step 3: Formal Submission (Month 12)
- Submit package to SAE via SAE Connect portal
- Pay review fee (estimated $8,000–15,000 based on scope)
- Assign Compliance Lead as primary point of contact for panel questions

### Step 4: Panel Response (Month 14–16 estimated)
- SAE panel reviews; issues either:
  - Letter of no objection (proceed to Phase 4)
  - List of required changes (remediate and resubmit)
- Timeline varies; Phase 4 launch is gated on receiving no-objection letter

## Risk: Timeline Dependency

SAE panel review takes 60–90 days after submission. If submission is in Month 12, earliest response is Month 14. Phase 4 target launch is Month 15. This is tight.

**Mitigation options:**
1. Submit earlier (Month 11) by compressing package assembly
2. Negotiate with SAE for expedited review (fee-based, not guaranteed)
3. Launch Phase 4 with "SAE review submitted — pending" status and document interim compliance posture (acceptable risk per outside counsel)
